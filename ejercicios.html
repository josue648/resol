Ejercicios Matemáticos

Ejercicios 6 al 10

  1. Defina la variable z como z = 4.5; luego evalúe:
    1. 0.4z⁴ + 3.1z² - 162.3z - 80.7
    2. (z³ - 23) / (∛(z² + 17.5))
  2. Defina la variable t como t = 3.2; luego evalúe:
    1. (1/2) e^(2t) - 3.8 ln(t³)
    2. (6t² + 6t - 2) / (t² - 1)
  3. Defina las variables x y y como x = 6.5 y y = 3.8; luego evalúe:
    1. ((x² + y²)^(2/3)) + (xy / (y - x))
    2. √(x + y) / (x - y)² + 2x² - xy²
  4. Defina las variables a, b, c y d como:
    c = 4.6, d = 1.7, a = c d², b = (c + a) / (c - d); luego evalúe:
    1. e^(d-b) + ∛(c + a - (ca)^d)
    2. d/c + (ct/b)² - c^d - a/b
  5. Dos identidades trigonométricas se dan por:
    1. cos²x - sin²x = 1 - 2sin²x
    2. (tan x) / (sin x - 2 tan x) = 1 / (cos x - 2)
    Para cada parte, verifique que la identidad es correcta calculando los valores de ambos lados de la ecuación, sustituyendo x = π / 10.
